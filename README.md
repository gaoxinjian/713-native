# 七幺三 - 旋风无敌小转盘

一个简单好用的微信小程序转盘工具，帮你解决选择困难症。

## 功能特点

- 🎯 **随机转盘** - 点击按钮，转盘随机旋转，帮你做决定
- 📝 **自定义选项** - 支持添加、编辑、删除转盘选项
- 🎨 **自动配色** - 每个选项自动生成美观的配色
- 💾 **本地存储** - 数据保存在本地，下次打开依然存在
- 🎴 **多转盘管理** - 支持创建多个转盘，随时切换

## 使用场景

- 今天吃什么？
- 谁来买单？
- 周末去哪玩？
- 任何需要随机选择的场景

## 项目结构

```
七幺三-原生版/
├── pages/
│   ├── index/          # 首页（转盘页面）
│   ├── list/           # 转盘列表页
│   └── edit/           # 编辑转盘页
├── utils/
│   └── storage.js      # 本地存储工具
├── app.js              # 应用入口
├── app.json            # 应用配置
└── app.wxss            # 全局样式
```

## 技术实现

- 使用微信小程序原生 Canvas API 绘制转盘
- Canvas 绘制完成后导出为图片，避免原生组件层级问题
- CSS transform 实现转盘旋转动画
- 本地存储使用 wx.setStorageSync

## 使用说明

1. 使用微信开发者工具打开项目
2. 点击"开始转"旋转转盘
3. 点击"切换转盘"管理多个转盘
4. 在列表页可以添加、编辑、删除转盘
